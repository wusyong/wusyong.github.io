Getting Started

We are going to install the Haskell Platform which is a all-in-one installer: https://www.haskell.org/platform/

What you installed:

  • GHC: Glasgow Haskell Compiler
    • ghci to start an interactive environment.
    • :l hello to load the file hello.rs, and :r to reload it.
  • Cabal build system: A build tool focused on dependency resolution and sources packages from Hackage
  • Stack tool: A build tool focused on curated snapshots and sources packages from Stackage
  • alex: Lexical analyzer generator
  • happy: Parser generator
